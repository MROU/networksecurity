    项目创新

    目的：充分发挥学生想象力，调动学生积极性。

    项目创新介绍

    无论你喜欢什么语言， java ，C ，php，jsp，asp，只要有创新的点子，
都可以提交到本页我的邮箱。如果通过审核，就可以组织本班同学
为一组，开发软件，老师将随时为你答疑。

    项目创新要求

    请在提交你的点子前，完成项目创新报告，对你的软件创新点进行分析。
如果你的创新点子没有通过，请不要灰心，我们将回互你的邮件，指出相应的优缺点
